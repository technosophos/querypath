Class QPTableTextData

Class Overview
Class Overview | Inherited Properties, Constants, and Methods | Properties | Method Summary | Methods Detail

Provides a table where all of the headers and data are treated as text data.

This provents marked-up data from being inserted into the DOM as elements. Instead, the text is escaped using {@see htmlentities()}.
