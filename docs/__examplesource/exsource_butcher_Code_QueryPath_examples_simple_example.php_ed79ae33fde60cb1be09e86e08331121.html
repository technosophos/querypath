Basic Example

  1. <?php
  2. /**
  3.  * Basic example of QueryPath usage.
  4.  *
  5.  * This two-line example exhibits basic use of QueryPath. It creates a new
  6.  * HTML document and adds the typical 'Hello World' text to the body. It then writes
  7.  * that information to standard out (which is flushed to a web browser in most cases.)
  8.  *
  9.  * The important methods covered here are {@link qp()}, which is the {@link QueryPath}
  10.  * factory function, {@link QueryPath::find()}, which is the primary searching
  11.  * function, and {@link QueryPath::writeHTML()}, which is a utility function.
  12.  *
  13.  * This file is fully explained in the official QueryPath tutorial, located
  14.  * at {@link https://fedorahosted.org/querypath/wiki/QueryPathTutorial}
  15.  *
  16.  * @package Examples
  17.  * @author M Butcher <matt@aleph-null.tv>
  18.  * @license LGPL The GNU Lesser GPL (LGPL) or an MIT-like license.
  19.  * @see qp()
  20.  * @see QueryPath::find()
  21.  * @see QueryPath::writeHTML()
  22.  * @see html.php
  23.  * @see https://fedorahosted.org/querypath/wiki/QueryPathTutorial The Official Tutorial
  24.  */
  25. require_once 'src/QueryPath/QueryPath.php';
  26. qp(QueryPath::HTML_STUB)->find('body')->text('Hello World')->writeHTML();
  27. ?>
